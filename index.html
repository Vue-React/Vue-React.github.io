Каталог на Vue 3, PHP + MySQL
Мультиязычный каталог (SPA) - админ-панель.
Приложение в процессе разработки, периодически обновляется. Вы можете посмотреть то, что готово на данный момент.
  • Frontend (SPA): Vue, Vuex, Vue Router, Axios
  • Backend: PHP + MySQL
  • Взаимодействие фронта и бэка по REST API (JSON)
  • Продвинутая JWT авторизация (в соответствии с Best Practices: 2 токена, передача Access Token в заголовках, RefreshToken в Cookie HttpOnly Secure, использование интерсепторов Axios для запросов и обработки ответов, настройка CORS политики для кросс доменных запросов и т.д.)
  • Сортировка, пагинация.
  • Мультиязычность реализована и на стороне клиента, и на стороне сервера (в БД).
  • Применялись: Composer, Postman, Firebase/PHP-JWT
  • Фронт размещен на Github Pages. Бэк на коммерческом хостинге.
